# Juntos para Sempre — Site (em construção)

Este repositório contém um site simples e responsivo para a clínica veterinária "Juntos para Sempre". O design utiliza um fundo degradê azul suave e contém seções: Sobre nós, Serviços e Contato (placeholders).

Arquivos incluídos:
- `index.html` — página principal
- `styles.css` — estilos, incluindo o degradê e componentes

Como usar
1. Clone o repositório:
   git clone https://github.com/jesus1204/veterin-rio-Juntos-para-Sempre.git

2. Copie/adicione os arquivos acima na raiz do repositório (ou no diretório `docs/` se preferir usar GitHub Pages a partir da pasta `docs`).

3. Para visualizar localmente, abra `index.html` no navegador ou use um servidor local:
   - Python 3: `python -m http.server 8000`
   - então acesse `http://localhost:8000`

Deploy (GitHub Pages)
- Opção 1: Ativar GitHub Pages para a branch `main` e a pasta `/ (root)` nas configurações do repositório.
- Opção 2: Colocar os arquivos em `docs/` e escolher a pasta `docs` nas configurações de Pages.

Personalização rápida
- Alterar textos em `index.html` (nome, contatos, serviços).
- Ajustar cores no `:root` de `styles.css` (variáveis CSS).
- Adicionar logo substituindo o texto da marca por uma imagem.
